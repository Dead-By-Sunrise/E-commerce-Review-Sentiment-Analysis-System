# 产品评论情感分析系统 - 中期答辩报告

---

## 一、项目背景与意义

### 1.1 研究背景

随着电子商务的快速发展，网络购物已成为主流消费方式。京东、天猫等平台上每天产生海量的用户评论，这些评论蕴含着丰富的用户反馈信息，对于消费者购买决策和商家产品改进具有重要价值。然而，人工逐一阅读分析评论费时费力，因此需要借助自然语言处理技术实现自动化情感分析。

### 1.2 研究目标

构建一个端到端的产品评论情感分析系统，实现：
- **数据采集**：自动从电商平台爬取商品评论
- **情感分析**：基于机器学习模型对评论进行情感分类
- **可视化展示**：通过丰富的图表直观展示分析结果

### 1.3 应用价值

| 面向对象 | 价值说明 |
|---------|---------|
| **消费者** | 快速了解产品口碑，辅助购买决策 |
| **商家/企业** | 自动化分析用户反馈，精准把握产品优缺点，为产品迭代和营销策略提供数据支撑 |
| **教学/学习** | 融合软件工程、机器学习、数据挖掘、数据可视化等多领域知识，全面锻炼综合能力 |

### 3.2 数据清洗模块 ✅ 已完成

**文件**：`data_clean_store.py` (538行)

**处理流程**：

```
原始JSON → HTML实体解码 → URL过滤 → Emoji去除 → 
文本规范化 → 情感标签生成 → 商品分类 → 幂等存储
```

**核心功能**：
1. **文本规范化**
   - HTML实体转义（&nbsp; → 空格）
   - 去除URL链接
   - Emoji表情过滤
   - 多余空白字符合并

2. **情感标签生成**
   ```
   评分 >= 4 → positive
   评分 == 3 → neutral  
   评分 <= 2 → negative
   ```

3. **商品分类**（基于关键词匹配8大品类）
   - 强规则优先：食品生鲜类（如"菠萝"、"茅台"）优先判定
   - 词元化匹配：使用jieba分词降低误判
   - 打分机制：优先命中更长、更具体的关键词

4. **数据存储**
   - SQLite数据库（幂等插入，SHA256去重）
   - 导出CSV供模型训练使用

**输出文件**：
- `goods_cleaned/comments.sqlite` - SQLite数据库
- `goods_cleaned/datasets/all_comments_clean.csv` - 合并CSV
- `goods_cleaned/manifest.json` - 数据清单

### 3.3 情感分析模块 ✅ 已完成

**文件**：`aspect_sentiment_pipeline.py` (712行)

**实现模型**：

| 模型 | 类型 | 特点 |
|-----|------|-----|
| **Lexicon** | 传统词典 | 基于正/负面词表+否定词处理，无需训练 |
| **TextCNN** | 深度学习 | 多尺度卷积核捕捉局部特征 |
| **BiLSTM** | 深度学习 | 双向LSTM捕捉上下文依赖 |
| **BERT** | 预训练语言模型 | 需安装transformers（可选） |

**模型架构详情**：

```
TextCNN:
- 词嵌入维度: 128
- 卷积核: (3, 4, 5) 三种尺寸
- 卷积核数量: 64
- Dropout: 0.3
- 优化器: Adam (lr=1e-3)

BiLSTM:
- 词嵌入维度: 128
- 隐藏层维度: 128
- 双向: True
- Dropout: 0.3
```

**数据集划分**：
- 训练集: 64%
- 验证集: 16%
- 测试集: 20%

**评估指标**：
- Accuracy（准确率）
- Precision_macro（宏精确率）
- Recall_macro（宏召回率）
- F1_macro（宏F1值）

**方面级情感分析**：

针对8大品类定义了不同的方面维度：

| 品类 | 方面维度 |
|-----|---------|
| 3C数码与家电 | 性能/功能、外观设计、安装/物流、质量/耐用性、交互/易用性 |
| 服饰鞋包与配饰 | 版型/尺码、舒适度、材质/质感、外观/款式、做工细节 |
| 美妆个护与健康 | 安全性/温和度、功效/效果、质地/肤感、包装/便携性、气味 |
| 家居生活与家具 | 环保/气味、尺寸/规格、材质/硬度、安装/售后、稳定性/耐用性 |
| 食品饮料与生鲜 | 口味/口感、新鲜度/保质期、物流/包装、分量/性价比、配料/健康度 |
| 母婴用品与玩具 | 安全性/材质、功能性/设计、适用性/年龄、清洁/便利性、正品/溯源 |
